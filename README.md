# csv-ed (CSV Editor)

`csv-ed` は、Python と PySide6 (Qt) を使用して構築された、Windows向けのモダンなGUI付CSVエディタアプリケーションです。
人間工学に基づいたダークテーマ/ライトテーマに対応し、複数のCSVファイルをタブや分割ウィンドウ（ドック）形式で直感的に並行編集できます。

## 主な機能

- **マルチドック & タブUI**: CSV編集グリッドはドッキング可能なウィンドウになっており、分割表示、タブ表示、フローティング表示など、好みに合わせたレイアウトが可能です。
- **左ペインのファイルブラウザ**: 左ペインにフォルダおよびCSVファイルの一覧を表示。ダブルクリックでファイルを開く、シングルクリックでプレビューといった操作を快適に行えます。
- **ツールバー**: 
  - `menu`: 新規作成、ファイルオープン、テーマ切り替え（ダーク/ライト）、終了処理。
  - `UpDir`: フォルダ階層を1つ上に戻る。
  - `save`: 編集中のCSVファイルを保存。
  - `設定`: デフォルトディレクトリや標準エンコーディング（UTF-8、Shift-JIS等）を設定。
- **設定の永続化**: ウィンドウ位置・サイズ、ドックのレイアウト状態、最後に開いていたディレクトリ、選択テーマ、エンコーディング等の情報は自動的に `%LOCALAPPDATA%\mh-csv-ed\config.toml` に保存され、次回起動時に復元されます。
- **グリッド編集機能**:
  - 行や列の挿入・削除、セルのインプレース編集。
  - 右クリックメニューからの行・列操作。
  - 変更検知による未保存警告機能。

## 必要条件

アプリケーションを動かすには、以下の環境が必要です。

- **Python 3.8+**
- 依存ライブラリ:
  - **PySide6**
  - **toml**

## セットアップと起動方法

### 1. 依存ライブラリのインストール

```bash
pip install PySide6 toml
```

### 2. アプリケーションの起動

リポジトリのルートディレクトリで以下のコマンドを実行します。

```bash
python main.py
```
