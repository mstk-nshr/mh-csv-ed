●マップ設定

  • BGM番号(ﾘｽﾄ読込)
    「[-2]イベントに任せる」に設定してください。
    それ以外の値を設定するとBGMが正しく再生されません！
  • BGMモードリスト
    • マップ移動時にそのBGMを再生させるか、BGMを消去させるか、イベントに任せるかを設定します。
  • マップ表示名
    • その文字列がゲーム上で表示されます。
  • マップのフォグ
    • そのマップのフォグを指定します。
  • フォグの合成方法
    • そのマップのフォグの合成方法を指定します。
  • フォグの不透明度
    • そのマップのフォグの不透明度を指定します。
  • マップの天候
    • そのマップの天候を指定します。
  • 雨/雪の傾き[0=真下]
    • 雨と雪の傾き具合を指定します。「0」で真下に降ります。
  • 雨が降るまでのﾌﾚｰﾑ数
    • フレーム数が「1」以上かつ小さいほど雨と雪が早く降ります。
  • 拡張遠景ID1～5
    • そのマップの遠景を最大5枚まで重ねて表示できます。
      また、画面サイズに応じて自動で拡大縮小されます。
      ただし、SDB「遠景画像」の以下の項目は無視されます。
      • X移動速度
      • Y移動速度
      • マップと位置連動する？
  • 拡張フォグID/合成方法/不透明度1～5
    • そのマップのフォグを最大5枚まで重ねて表示できます。
      また、画面サイズに応じて自動で拡大縮小されます。
      合成方法や不透明度も個別に設定できます。
      ただし、SDB「遠景画像」の以下の項目は無視されます。
      • X移動速度
      • Y移動速度
      • マップと位置連動する？

●BGMリスト

  • 表示名
    • サウンドテストで表示する曲名を設定します。

●文字列変数名/通常変数名/予備変数1～9

  • 周回時に値を引き継ぐ？
    • 周回時に値を引き継ぐかを設定します。

●位置設定リスト

  • マップID
    • マップIDです。
  • X（Y）座標
    • X（Y）座標を指定します。
  • テレポ条件変数
    • テレポートを禁止するか、無条件でテレポートを許可するか、
      テストプレイ時のみテレポートを許可するか、通常変数をテレポートの条件にするかを設定します。
  • 海上（飛行）船テレポ位置
    • 海上（飛行）船のテレポート位置を設定します。
      SDB「位置設定」からデータIDで指定します。
  • 表示名
    • テレポート先の選択画面で表示される表示名です。

●顔グラフィック名

  • (データ名)、表示名
    • メッセージウィンドウの名前表示ウィンドウに表示する名前です。
      「@」を含めるとそれ以降の文字を表示しません。
      • 例:「ウルファール@笑顔」の場合は「ウルファール」と表示される
  • 顔画像ファイル、顔画像拡大率
    • メッセージウィンドウで表示する立ち絵(顔画像)について設定します。
      顔画像ファイルが空欄の場合は表示しません。
  • 敵画像代替表示
    • SDB顔画像の代わりに敵画像をメッセージに表示します。
  • ﾒｯｾｰｼﾞｳｨﾝﾄﾞｳﾍﾞｰｽ色Ｒ/Ｇ/Ｂ
    • メッセージウィンドウの色を変更します。
