利用規約

  • 全ての素材について、二次配布や改造物の配布に関しては
    無償であり、見返りを求めない場合のみ許可します！
  • 全ての素材について、改造に関しては
    悪意を持った改造を禁止します！
  • Mr.Hが作った素材、またはMr.Hが改変した素材に関しては、商用利用に関しては以下の条件で商用利用とみなします！
    • 広告、または課金コンテンツが含まれている場合
    • 商用展開のあるコンテストに参加する場合
  • 全ての素材について、以下の行為を禁止しています！
    • 素材単体での販売
    • 著作権/肖像権侵害
    • 犯罪行為
    • 不謹慎な行為
    • 政治利用
    • 事件を題材にしたコンテンツ
    • 新しい宗教を題材にしたコンテンツ
      （神話や昔話を題材にするのは可）
    • 差別的行為
    • AI学習リソースへの利用
    • 著作権登録
      • 「著作権管理事業者」または「著作権等管理団体」への登録を禁じます！
      • ベルヌ条約への加盟の折り合いが複雑な国において、一部方式主義の慣例を引き継いでいる国における著作権登録も禁止します
    • 商標登録
      • 「著作権管理事業者」または「著作権等管理団体」への登録を禁じます！
      • ベルヌ条約への加盟の折り合いが複雑な国において、一部方式主義の慣例を引き継いでいる国における商標登録も禁止します
  • Mr.Hが作成した全ての画像/音楽素材について
    AIの学習リソースへの使用を禁止します！
  • 素材元からお借りした全ての画像/音楽素材について素材元から特記が無い限り
    AIの学習リソースへの使用を禁止します！
  • 他の人/サイトの素材を追加して利用する際
    AIによって生成された素材を追加する事を禁止します！
    (例:それぞれの素材制作者がAIを使わずに自力で作った素材である事)
  • AI生成素材を使用する際は「適切な権利がある素材」で学習されたモデルやサービス以外で生成したAI生成素材の使用を禁止します。
    以下はこの基準です。
    (※予告なく基準を変更する場合があります！)
    • Adobeの「Adobe Firefly」や、Stable Diffusion用の「Mitsua Diffusion」モデルのような
      「パブリックドメインやCC0の素材、許諾ありの素材のみで学習されたAI生成素材」(オプトイン制)
      のAIで生成した素材のみ使用可
    • Stable Diffusion用の基本モデルを含むほぼ全てのモデル、NovelAI、Midjourney系、Dall-E(Bing、Windows Copilot、ChatGPT)などの、学習元データが明示されていない、または「モデル/サービス提供者が権利を持たない画像」で学習されたAIで生成した素材は使用禁止
  • 無料高機能RPGシステムMr.Hのコモンイベントおよびデータベース部分(処理内容)について、第三者の手によるウディタ以外のプログラムへの移植および、ゲーム制作ソフトとしての配布を、クレジット表記を行うことを条件に許可します。
    ただし、素材単体での販売や自作発言は禁止とします。
    また、ゲーム制作ソフトとして配布する場合は、以下の条件を満たすことを必須とします。
    • 完全無料であること
    • 処理内容などがオープンソースで公開されていること
    • ソースコードの二次配布、改造、および改造物の配布を許可すること
    • 広告を一切表示しないこと
    • 課金要素を設けないこと
    • 見返りを求めないこと
    • 同梱素材の規約を守ること
      (特に、二次配布禁止の素材は同梱不可)

詳しくは下記の利用規約を参照してください。
