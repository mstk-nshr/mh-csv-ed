■スキル/アイテム関連設定

●使用ｽｷﾙ設定1

  • 敵を蘇生させるスキルの追加
    「┏ 効果対象 」を「ｲﾍﾞﾝﾄ呼出[ﾊﾞﾄﾙのみ]」に設定したうえで、「(あれば)┗ｲﾍﾞﾝﾄ番号指定」に以下を入れてください
    後は敵キャラにそのスキルを使用させるだけです。
    • -500000:戦闘不能になった敵キャラの中からランダムで一人蘇生させます。
    • -500001:戦闘不能になった敵キャラを全員蘇生させます。
  • 敵を蘇生させるスキルの追加
    後は敵キャラにそのスキルを使用させるだけです。
    • 「[戦]発動前コモン呼び出し」を「[-2]<敵ｸﾞﾙｰﾌﾟ全体蘇生>」に指定
      • 戦闘不能になった敵キャラを全員蘇生させます。
    • 「発動後コモン呼び出し」を「[-2]<敵ｸﾞﾙｰﾌﾟ全体蘇生>」に指定
      • 戦闘不能になった敵キャラを全員蘇生させます。
  • 敵が仲間を呼ぶスキルを追加
    「┏ 効果対象 」を「[102]仲間を呼ぶ[敵のみ]」に設定したうえで、「(あれば)┗ｲﾍﾞﾝﾄ番号指定」に敵キャラのIDを入れてください
    奇形魚を呼び出す場合は「(あれば)┗ｲﾍﾞﾝﾄ番号指定」に「5」を設定してください
    「(あれば)┗ｲﾍﾞﾝﾄ番号指定」が「-1」以下の場合は自分と同じ敵キャラを呼び出します。
    味方がそのスキルを使用した場合は何もしません。
  • 戦闘フィールドを操作するスキルを追加
    「┏ 効果対象 」を「[104]戦闘ﾌｨｰﾙﾄﾞ変化」に設定してください。
    戦闘フィールドは敵味方全体に効果があります。
    ※「付与するｽﾃｰﾀｽ状態2～3」「自由状態付与番号」では動作しません！
    • 付与するｽﾃｰﾀｽ状態1
      • 1以下
        • 戦闘フィールドを消滅させます。
      • 0～9999
        • 戦闘フィールドを追加します。別の戦闘フィールドがあった場合は上書きします。
  • 召喚獣を追加
    「┏ 効果対象 」を以下のいずれかに設定してください。
    次に、「(あれば)┗ｲﾍﾞﾝﾄ番号指定」に任意の召喚獣のIDを指定します。
    • [105][自分]召喚獣召喚
      • 自分に対して召喚獣を召喚させます。
    • [106][味方]召喚獣召喚
      • ターゲットで選択した味方に対して召喚獣を召喚させます。
    • [107][敵]召喚獣召喚
      • ターゲットで選択した敵に対して召喚獣を召喚させます。
  • 移動時のテレポートスキルを追加
    「┏ 効果対象 」を「[200]ﾃﾚﾎﾟｰﾄ[ﾏｯﾌﾟのみ]」に設定してください。
    テレポートが禁止されているまたはエスケープが許可されているまたは戦闘時の場合は使用できません。
  • 移動時のエスケープスキルを追加
    「┏ 効果対象 」を「[201]ｴｽｹｰﾌﾟ[ﾏｯﾌﾟのみ]」に設定してください。
    エスケープが禁止されているまたは戦闘時の場合は使用できません。
  • 移動時のエンカウント率操作スキルを追加
    「┏ 効果対象 」を「[202]ｴﾝｶｳﾝﾄ率操作[ﾏｯﾌﾟのみ]」に設定してください。
    戦闘時の場合は使用できません。
    「(あれば)┗ｲﾍﾞﾝﾄ番号指定」に以下の数値を入れてください。
    • 持続歩数を1～99999までの値かつそのままの値(1倍)で代入してください。
    • エンカウント率を1～9999までの値かつ100000倍の値で加算してください。
    • 例:5001000
      • 1000歩の間、エンカウント率が50％(半減)
    • 例:2500750
      • 750歩の間、エンカウント率が25％(1/4倍)
    • 例:20001500
      • 1500歩の間、エンカウント率が200％(2倍)
    • 例:500
      • 500歩の間、エンカウント率が0％(無効)
  • 蘇生用のスキルを作る場合は、対象を味方単体か味方全体にしたうえで、「成功罹患必須状態」を「[1][蘇生]死者のみ」か「[2][蘇生]生存者＆死者」にしてください。
    「回復するｽﾃｰﾀｽ状態」に何も設定されていなくても戦闘不能が解除されます。
    デフォルトのRPG基本システムでは蘇生用のスキルが生存者にも有効です。
    「[1][蘇生]死者のみ」にすると、生存者には効かず、死者にのみ有効な蘇生スキルになります。
    「[2][蘇生]生存者＆死者」にすると、生存者にも死者にも有効な蘇生スキルになります。

  • 名前
    • スキルの名前です。
  • タグ:カスタム
    • 上級者向けの設定をするための機能です。
      カスタムタグについては別紙を参照ください。
  • 説明
    • スキルの説明文です。
  • アイコン
    • アイコンをUDB「アイコン画像設定」からデータIDで指定します。
  • 効果対象
    • メッセージのみ
      • メッセージを表示するだけで何もしません。
    • 味方一人、自分自身のみ、味方全員、味方一人ランダム、自分以外の味方一人、自分以外の味方全体
      • 味方に回復技を使用します。
    • 敵一体、敵全体、敵一体にランダム、敵味方全体、敵味方ﾗﾝﾀﾞﾑ、自分以外敵味方全体、自分含む敵全体
      • 攻撃技を使用します。
    • [自分/味方/敵]召喚獣召喚
      • 召喚獣を召喚します。
    • 敵一体の行動をものまね
      • 選択した敵一体が最後に使用したスキルを使用します。
        ものまね可能なスキルが無い場合は発動に失敗します。
    • 直前の行動をものまね
      • この戦闘で最後に使用されたスキルを使用します。
        ものまね可能なスキルが無い場合は発動に失敗します。
    • 戦闘ﾌｨｰﾙﾄﾞ変化
      • 戦闘フィールドを変化させます。
        戦闘フィールドは敵味方全員に有効です。
    • [味方]敵一体を捕獲
      • 選択した敵一体を捕獲します。
        捕獲した敵の捕獲スキルが使用可能になります。
        すでに捕獲スキルがある場合は使用できません。
        捕獲スキルが設定されていない敵は捕まえる事ができません。
        捕獲した敵は戦利品を得られず、図鑑にも登録されずクエストの討伐数にもカウントされません。
        味方専用のスキルです。
    • [味方]捕獲したのを放つ
      • 捕獲スキルを発動します。
        敵を捕獲していない場合は使用できません。
        使用すると捕獲スキルは消滅します。
        味方専用のスキルです。
    • [味方]敵一人のｽｷﾙ覚える
      • 選択した敵一人のスキルを覚えます。
        覚えるスキルはデフォルトAIで指定したものです。
        ものまね禁止のスキルとスキルでない行動は覚えません。
        敵のスキルを覚えた状態で別の敵のスキルを覚えると前の敵から覚えたスキルは消滅し新たな敵から覚えたスキルで上書きされます。
        また、敵から覚えたスキルは戦闘が終わると消滅します。
        味方専用のスキルです。
    • [味方]敵一人を食べる
      • 選択した敵一体を食べます。
        食べた敵の食べるスキルを永久的に覚えます。
        食べるスキルが設定されていない敵は食べる事ができません。
        食べた敵は戦利品を得られず、図鑑にも登録されずクエストの討伐数にもカウントされません。
        味方専用のスキルです。
    • [味方]敵一人/全体/ランダムｱｲﾃﾑ変化
      • 敵一人/全体/ランダムをアイテムに変化させます。
        味方専用のスキルです。
    • [味方]敵一人/全体/ランダム威嚇
      • 敵一人/全体/ランダムを威嚇させて逃走させます。
        味方専用のスキルです。
    • [味方]スキル側逃走
      • 戦闘から逃走します。
        逃げられない戦闘では使用できません。
        味方専用のスキルです。
    • 仲間を呼ぶ[敵のみ]
      • 仲間を呼びます。
        敵専用のスキルです。
    • 他の敵ｷｬﾗに変身[敵のみ]
      • 他の敵キャラに変身します。
        敵専用のスキルです。
    • お金を奪う[敵のみ]
      • 味方側からお金を奪います。
        盗む金の量は「最低発動回数」～「最高発動回数」となります。
        敵専用のスキルです。
    • ｲﾍﾞﾝﾄ呼出[ﾊﾞﾄﾙ/ﾏｯﾌﾟのみ]
      • 戦闘時/マップ時にイベントを呼び出します。
        戦闘/マップ専用のスキルです。
    • テレポート/エスケープ[ﾏｯﾌﾟのみ]
      • テレポート/エスケープします。
        マップ専用のスキルです。
    • エンカウント率操作[ﾏｯﾌﾟのみ]
      • エンカウント率を操作します。
        マップ専用のスキルです。
  • (あれば)┗ｲﾍﾞﾝﾄ番号指定
    • 呼び出し対象のイベントIDまたは召喚対象の召喚獣などはここで指定します。
  • 対象全体化条件
    • 設定された条件を満たすと、スキルの対象が全体化されます。
      UDB「汎用条件設定1」からデータIDで指定します。
  • ┏ 消費するものは？
    • 無料高機能RPGシステムMr.Hでは、「スキル消費コスト設定1」からデフォルトAIの行動内容を設定します。
      その参照IDです。
  • 発動タイミング
    • 使用できるシーンと、発動時ターンを消費しないかどうかと、テストプレイ時限定かどうかを設定します。
  • 武器の影響
    • 武器に影響させるかのフラグです。
  • 成功率[%] [-1:命中依存]
    • スキルの成功率です。
      • -1以下
        • 発動者の命中率が成功率となります。
  • 武器依存時命中補正±%
    • 武器依存時の命中率の補正割合です。
  • クリティカル率[%] (-1:ﾅｼ)
    • クリティカルが発生する確率です。
      発動者のクリティカル率がスキルのクリティカル率に加算されます。
      • -1以下
        • クリティカルを発生させません。
  • 最低、最大発動回数
    • 発動回数を設定します。
  • 追加発動用スキル種別
    • 追加発動用のスキル種別です。
  • 優先度
    • スキルの優先度です。
      値を大きくすると先に行動しやすくなります。
      ATB制戦闘ではほぼ意味の無い設定項目です。
  • 使用時文章[移動](人名~～失敗時文章[(対象)～]
    • 移動時、戦闘時のスキル使用時、命中失敗時のメッセージです。
      命中失敗時のメッセージを空欄にするとデフォルトの文字列が表示されます。
  • ダメージ/命中タイプ
    • 命中タイプを必中、物理、魔法の中から設定します。
      回避率やスキル無効率に影響するかも設定します。
  • 成功罹患必須状態
    • 蘇生スキルや治療スキルの対象が、生存している、もしくは該当の状態にかかってないと命中に失敗するようにします。
      純粋に蘇生スキルを設定する場合は「死者のみ」に
      治療、ディスペルを設定する場合は「解除可能状態罹患者のみ」に設定します。
  • 庇護、防御無視設定
    • 庇護を無視するか、防御状態を無視するかを設定します。
  • スキル命中条件
    • 設定された条件を満たさないとスキルの命中が失敗します。
      UDB「汎用条件設定1」からデータIDで指定します。
  • スキル反射条件
    • 設定された条件を満たすとスキルが反射されます。
      UDB「汎用条件設定1」からデータIDで指定します。
  • 与ダメージ率上書き
    • 与ダメージ率の乗算方法を上書きします。
  • ダメージ計算番号
    • 無料高機能RPGシステムMr.Hでは、「汎用ダメージ計算1」からスキルのダメージ量を設定します。
      その参照IDです。
  • アニメーション1～3
    • スキルの戦闘アニメです。
      発動回数だけ再生されます。
      UDB「戦闘アニメセット設定」からデータIDで指定します。
  • 発動、対象側発動時アニメーション
    • スキルの戦闘アニメです。
      発動後の1回だけ再生されます。
      UDB「戦闘アニメセット設定」からデータIDで指定します。
  • 付与属性
    • 無料高機能RPGシステムMr.Hでは、「属性ﾘｽﾄ設定1」からスキルの属性を設定します。
      その参照IDです。
  • 付与解除状態
    • 無料高機能RPGシステムMr.Hでは、「状態付与解除設定1」からスキルの状態付与と状態解除を設定します。
      その参照IDです。
  • スキルタイプ
    • スキルタイプです。
      UDB「ｽｷﾙﾀｲﾌﾟ設定」からデータIDで指定します。
  • スキル表示、使用条件
    • 設定された条件を満たさないとスキルが表示、使用できません。
      UDB「汎用条件設定1」からデータIDで指定します。
  • スキルセット不要
    • スキルをセットせずにスキルを使用できるかを設定します。
      スキル選択制のみ有効な項目です。
  • 必要スキル設定値
    • スキルをセットするのに必要なポイントです。
      スキル選択制のみ有効な項目です。
  • ｽｷﾙｾｯﾄ禁止ﾀｸﾞ
    • 重複したタグのスキルをセットできないようにします。
      スキル選択制のみ有効な項目です。
  • 与えた効果を自分に返す？、何に返す？
    • スキルのダメージ量を発動者への反動とするか、相手に与えたダメージを吸収して回復するかを設定します。
  • セルフ状態付与、解除
    • 発動者に状態を付与、解除します。
  • 発動者行動消去
    • 発動時、発動者のその後の行動が消去されます。
      主に、複数回行動する敵の大技用に使用します。
  • 予約スキル、発動までのターン
    • スキルを使用した一定ターン後、設定されたスキルを発動します。
      • 例:打ち上げた炎が3ターン後に降ってくるなど
  • ｸｰﾙﾀｲﾑﾀｰﾝ数、開始時ｸｰﾙﾀｲﾑ、ｸｰﾙﾀｲﾑｸﾞﾙｰﾌﾟ
    • クールタイムのターン数について設定します。
      個別で適用するか一括で適用するかも設定します。
  • 盗み発生率、成功補正値
    • 盗むスキルについて設定します。
  • 優先度増減
    • ターゲットのそのターン内の行動の優先度を増減します。
      ATB制戦闘ではほぼ意味の無い設定項目です。
  • 庇護使用、ターン数、HP条件、ダメージ防御値
    • 庇護スキルについて設定します。
  • 経験値、金額、ドロップ率操作[-1:ﾅｼ]
    • その戦闘の獲得経験値、金額、ドロップ率の割合を操作します。
      スキルの対象が、戦闘フィールドになってる必要があります。
  • 発動回数だけATB消費
    • スキルの発動回数分、ATBを消費させます。
      ATB制戦闘でのみ有効な項目です。
  • ATB命中失敗時再販定数
    • 命中失敗時、ATBゲージをもう1回消費して命中判定を再判定する回数です。
      ATB制戦闘でのみ有効な項目です。
  • ATB発動時消費ATB
    • 発動時、ATBゲージを消費する割合です。
      ATB制戦闘でのみ有効な項目です。
  • ATB増加固定値
    • 発動時、ATBゲージの増加に使用される敏捷性の値を上書きします。
      • -１以下
        • 発動者の敏捷性が、ターン経過の敏捷性になります。
      • -10000
        • 発動者の攻撃力が、ターン経過の敏捷性になります。
      • -10001
        • 発動者の防御力が、ターン経過の敏捷性になります。
      • -10002
        • 発動者の魔法力が、ターン経過の敏捷性になります。
      • -10003
        • 発動者の魔法防御が、ターン経過の敏捷性になります。
      • -10010
        • 発動者のカスタム1が、ターン経過の敏捷性になります。
      • -10011
        • 発動者のカスタム2が、ターン経過の敏捷性になります。
      • -10012
        • 発動者のカスタム3が、ターン経過の敏捷性になります。
      • -10013
        • 発動者のカスタム4が、ターン経過の敏捷性になります。
      • -10014
        • 発動者のカスタム5が、ターン経過の敏捷性になります。
      ATB制戦闘でのみ有効な項目です。
  • 命中率TP増減
    • 命中時、TPを増減させます。
  • 作戦型AIで使用禁止
    • 作戦型AI(オートコマンド)で使用できないようにします。
  • ジャマー、ものまね禁止
    • ジャマーやものまねの対象にならなくするかを設定します。
  • スキル追撃番号、スキル追撃率
    • スキル追撃について設定します。
  • 行動回数追加フラグ
    • 敵撃破や弱点を突いた時、クリティカルが出た時に行動回数を追加するかを設定します。
  • QTE使用
