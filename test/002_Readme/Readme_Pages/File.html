同梱ファイルについて

「無料高機能RPGシステムMr.H」のダウンロード先に格納されたファイルの概要や
本マニュアルで説明するエディションにおける、ダウンロードした際に同梱されているファイルやツールについて
説明しています。

ダウンロード先のファイル概要

①無料高機能RPGシステムMr.H　通常版
　コモンイベント「無料高機能RPGシステムMr.H」の通常版エディションになります。
　RPGを作るうえで必要最小限のサンプルデータおよびサンプル素材が同梱されています。

②無料高機能RPGシステムMr.H　ケモプレコラボ版
　コモンイベント「無料高機能RPGシステムMr.H」のケモプレコラボ版エディションになります。
　ケモプレデーションゲームス様が作ったゲームがサンプルゲームとして同梱されています。

③無料高機能RPGシステムMr.H　空データ
　コモンイベント「無料高機能RPGシステムMr.H」の空データになります。
　動かすのに必要最低限のデータや素材以外は入っていません。

④無料高機能RPGシステムMr.H　追加素材
　コモンイベント「無料高機能RPGシステムMr.H」（通常版／ケモプレコラボ版）では採用されていないコモンイベント及び
素材が含まれています。
　必要に応じてコモンイベントの追加や、素材を追加していきます。

⑤無料高機能RPGシステムMr.H　Android版サンプルゲーム
　コモンイベント「無料高機能RPGシステムMr.H」のAndroid版のサンプルゲームとなります
　Android上でどう動くかを確認できます。

⑥無料高機能RPGシステムMr.H　V2用RTP
　コモンイベント「無料高機能RPGシステムMr.H」のV2系で作られたゲームのRTPとなります。
　現在はこれを導入せずともプレイ可能ですがサポートの兼ね合いで残してます。

⑦実行用EXEファイル
　コモンイベント「無料高機能RPGシステムMr.H」のゲームとエディターを動かすのに必要なEXEファイルです。
　本来なら①②に含めて同梱するつもりでしたが、ウィルスの誤検出の関係上、別で配布しています。
