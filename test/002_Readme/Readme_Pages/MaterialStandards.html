●歩行画像

UDB「歩行画像設定」で設定

  • 画像のサイズはウディタデフォルトと同一の規格

●顔画像

UDB「顔画像設定」で設定

  • 画像のサイズは正方形の画像なら任意
    画面サイズに応じてリサイズ対応

●敵キャラ画像

UDB「敵キャラ画像設定」で設定

  • 画像のサイズは任意
    拡大率を調整可
    画面サイズに応じてリサイズ対応

●アイコン画像

UDB「アイコン画像設定」で設定

  • 画像のサイズは640x480の画面基準で24ピクセルの正方形の画像推奨
    1個の画像で、横や縦に複数個のアイコンが並んでる画像にも対応
    画面サイズに応じてリサイズ対応

●戦闘背景

UDB「戦闘背景設定」で設定

  • 画像のサイズは任意
    自動でリサイズ対応
    2枚のレイヤーの画像を設定

●戦闘ウィンドウ画像

UDB「戦闘ウィンドウ設定」で設定

  • メインウィンドウ/ウィンドウカバー
    • 画像のサイズは320x240の画面基準で80x60ピクセル推奨
      画面サイズに応じてリサイズ対応
  • カーソル/エフェクト
    • 画像のサイズは320x240の画面基準で80x60ピクセル推奨
      画面サイズに応じてリサイズ対応
      通常表示か加算表示を選択可

●MONリングコマンド画像

UDB「MONリングコマンド」で設定

  • 画像のサイズは正方形の画像なら任意
    表示サイズはUDB側で設定
    自動でリサイズ対応

●マスク画像

UDB「マスク画像設定1」で設定

  • 画像のサイズは任意
    画面サイズに応じてリサイズ対応

●カットイン

UDB「カットイン用設定」で設定

  • エフェクト用画像
    • 画像のサイズは正方形なら任意
      自動でリサイズ対応
  • 効果音を設定する項目に関しては効果音の規格とUDB設定方法を参照してください

●戦闘アニメーション

UDB「戦闘アニメセット設定」で設定

  • UDB「単体アニメ設定」で設定したアニメ画像を設定

UDB「単体アニメ設定」で設定

  • 画像の規格に関して
    横サイズは「横分割数×任意」
    縦サイズは「縦分割数×任意」
    です
    画面サイズに応じてリサイズ対応
  • 効果音を設定する項目に関しては効果音の規格とUDB設定方法を参照してください

●サイドビュー武器画像

UDB「装備タイプ設定」で設定

  • 画像の規格に関して
    横サイズは「横分割数×任意」
    縦サイズは「縦分割数×任意」
    です
    画面サイズに応じてリサイズ対応

●実績ごほうび画像

UDB「実績設定」で設定

  • 画像のサイズは任意
    画面サイズに応じてリサイズ対応

●旧版スタッフロール画像

UDB「スタッフロール1」で設定

  • 背景/1枚絵
    • 画像のサイズは任意
      自動でリサイズ対応

●タイトル画面

UDB「タイトルUI変更」で設定

  • 遠景
    • 画像のサイズは任意
      自動でリサイズ対応
      2枚のレイヤーの画像を設定
  • メインタイトル/サブタイトル/＊＊＊＊(コマンド名)画像名
    • 画像のサイズは任意
      画面サイズに応じてリサイズ対応

●メニュー画面

UDB「メニューUI変更」で設定

  • 各種ウィンドウ/カーソル画像
    • 縦横3分割の画像を使用

●システム画像/効果音

UDB「ｼｽﾃﾑﾌｧｲﾙ設定1」で設定

  • 起動時ロード背景
    • 画像のサイズは任意
      自動でリサイズ対応
  • ローディング画像ファイル
    • 画像のサイズは任意
      画面サイズに応じてリサイズ対応
  • ウィンドウ画像ファイル/カーソル画像ファイル/メッセージウィンドウベース画像/スクロールバー画像
    • 縦横3分割の画像を使用
  • マウス矢印画像
    • 画像のサイズは任意
      自動でリサイズ対応
  • タッチUI画像
    • 320x240の画面基準で縦横20ピクセルの画像を使用
      UIの配置は以下の通り
      下左右上下
      (二重)      		左
      (二重)      		右
      (二重)      		上
      (二重)
      AボタンBボタンXボタンYボタンメニューヘルプ会話ログミニマップ
      前ページ
      (キャラクター)      		次ページ
      (キャラクター)      		前ページ
      (アイテム分類)      		次ページ
      (アイテム分類)      		対象単体化対象全体化データ削除(不使用)
      画面サイズに応じてリサイズ対応
  • ロゴ画像ファイル
    • 画像のサイズは任意
      画面サイズに応じてリサイズ対応
  • シナリオ画面背景1～2
    • 画像のサイズは任意
      2枚のレイヤーの画像を設定
      自動でリサイズ対応
  • シナリオ画面タイトルロゴ画像
    • 画像のサイズは任意
      画面サイズに応じてリサイズ対応
  • 戦闘背景カバー
    • 画像のサイズは任意
      自動でリサイズ対応
  • 状態アイコン数字画像
    • 画像のサイズは640x480の画面基準で24ピクセルの正方形の画像推奨
      1つの画像で横に11枚のアイコンを並べた画像を使用
      状態アイコンの上に表示される
      アイコンの並び順は以下の通り
      12345678910～∞
      画面サイズに応じてリサイズ対応
  • 敵カーソル画像
    • 横と縦のサイズは任意
      横分割数は2とする
      画面サイズに応じてリサイズ対応
  • 敵ターゲット画像/敵選択弱点画像
    • 横と縦のサイズは任意
      横分割数に関しては、「横サイズ÷縦サイズ」とする
      画面サイズに応じてリサイズ対応
  • 戦闘時影グラフィック
    • 画像のサイズは任意で
      1コマの画像を使用
      画面サイズに応じてリサイズ対応
  • 汎用QTE画像
    • 320x240の画面基準で縦横10ピクセルの画像を使用
      1つの画像で横に6枚、縦に2枚のアイコンを並べた画像を使用
      1段目はカウント前、2段目はカウント後
      それぞれの段は以下の規格となる
      待機
      (経過前)      		任意ボタン
      (経過前)      		下
      (経過前)      		左
      (経過前)      		右
      (経過前)      		上
      (経過前)
      待機
      (経過後)      		任意ボタン
      (経過後)      		下
      (経過後)      		左
      (経過後)      		右
      (経過後)      		上
      (経過後)
      画面サイズに応じてリサイズ対応
  • 効果音を設定する項目に関しては効果音の規格とUDB設定方法を参照してください

UDB「ｼﾅﾘｵ設定1」で設定

  • ゲームオーバー画像ファイル
    • 画像のサイズは任意
      自動でリサイズ対応
  • コマンド選択画像
    • コマンド選択スタイルをロールにする場合
      • 画像のサイズは320x240の画面基準で72x18ピクセルを使用
        画面サイズに応じてリサイズ対応
    • コマンド選択スタイルをスプレッドにする場合
      • 画像のサイズは320x240の画面基準で80x38ピクセルを使用
        画面サイズに応じてリサイズ対応
    • コマンド選択スタイルをセンタースプレッドにする場合
      • 画像のサイズは320x240の画面基準で130x50ピクセルを使用
        画面サイズに応じてリサイズ対応
  • 海上/飛行船画像
    • 画像のサイズはウディタデフォルトと同一のキャラチップの規格

●有料版専用画像/効果音

UDB「[P]プロ版関連設定」で設定

  • スクショアルバム背景
    • 画像のサイズは任意
      自動でリサイズ対応
  • スクショカーソル
    • 縦横3分割の画像を使用
  • スクショなし画像
    • 画像のサイズは任意
      自動でリサイズ対応
  • 効果音を設定する項目に関しては効果音の規格とUDB設定方法を参照してください

●UDB内効果音設定項目

  • 以下の文字列を設定します
    • 効果音のファイル名(改行)
      音量(改行)
      ピッチ

●UDB内テキスト設定項目

  • テキストファイルの文字コードは必ず
    「UTF-8(BOM付き)」
    にしてください！
    それ以外の文字コードだと必ず文字化けします！

●立ち絵

SDB「顔グラフィック名」で設定

  • 画像のサイズは任意
    画面サイズに応じてリサイズ対応

●マップ背景

SDB「遠景画像」で設定)

  • 画像のサイズは画面サイズ分以上のピクセル数

●共通の画像の規格

  • png、jpg、bmpに対応
    pngを強く推奨
    bmpは非推奨

●共通のBGMの規格

  • mid、ogg、mp3、wavに対応
    oggを強く推奨
    wavは非推奨
    (midの場合はたまに音楽が鳴らない)

●共通の効果音の規格

  • ogg、mp3、wavに対応
    • oggを強く推奨
      mp3は非推奨

●3D関連画像

  • アクション画像
    • 画像のサイズは任意
      横分割数に関しては、3とする
      以下の規格となる
      アクションメニューログ
      画面サイズに応じてリサイズ対応
  • 3D用のレーダー画像
    • 画像のサイズは任意
      横分割数に関しては4、縦分割数に関しては3とする
      1段目はオブジェクトの属性
      2段目は自身の向き
      3段目は方位となり
      それぞれ以下の規格となる
      透過通行可能通行不能EV
      上左下右
      北東南西
      画面サイズに応じてリサイズ対応
  • 3D用のコンパス画像
    • 画像のサイズは任意
      画面サイズに応じてリサイズ対応

●追加素材の2Dミニマップ画像

  • 横分割数に関しては4、縦分割数に関しては3とする
    縦分割2つ目までは3D用のミニマップと同じ。
    3つ目から10個目は、イベントに使用され、
    イベントの向きと横分割何番目の関係は、縦2分割目と同じです。
