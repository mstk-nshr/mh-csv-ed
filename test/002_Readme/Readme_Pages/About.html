無料高機能RPGシステムMr.H とは？

rute.氏の基本システムAdvancedをMr.H氏が大幅に強化した大規模統合型コモンです。

基本システムAdvancedの他に、Lucifa氏の３Dダンジョン作成・表示コモンなど様々なコモンがデフォルトで導入されています。

デバッガーにリーフスカイ氏を加え、さらに合作コモンウディタリオンからの機能移植などをケモプレデーションゲームスが受け持っています。

制作者のこんな悩み、ありませんか？

ゲームの大規模なアップデートができない…
　→ 主人公の能力値を後から修正可能！長期運用にも対応！

ブラウザやマウス操作に対応させたい…
　→ マルチプラットフォーム対応！ブラウザウディタ・8ボタン・マウス操作もOK！

戦闘システムをもっと充実させたい…
　→ サイドビュー・ATB・オートコマンドなど、臨場感ある戦闘が可能！

演出をもっと強化したい…
　→ 戦闘開始演出・クイック通知・あらすじ/ヘルプ機能などで物語を彩れる！

コモンイベントの組み合わせが大変…
　→ これ一本で、技能・装備・状態・敵AI・UIまで全部統合済み！

さらに、こんな機能も搭載！

錬成装備・二刀流・自動装備などで装備を自由にカスタマイズ

TP消費・クールタイム・召喚獣などで技能の戦略性が大幅UP

職業システム・アイテム個別制・戦闘難易度・強くてニューゲームなど、システム面も充実

セーブ画面の演出強化＆プロ版ではマップ背景表示やセーブデータ消去も対応

解像度に合わせたUI調整＆プロ版で自由指定可能
